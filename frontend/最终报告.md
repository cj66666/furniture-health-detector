## 🚀 微信小程序使用建议

### 数据索引优化
```javascript
// 按类别建立索引
const furnitureByCategory = {}
furnitureData.forEach(item => {
  const cat = item.category
  if (!furnitureByCategory[cat]) {
    furnitureByCategory[cat] = []
  }
  furnitureByCategory[cat].push(item)
})
```

### 风险评分系统
```javascript
function calculateRiskScore(item) {
  let score = 100
  item.data.risk_points.forEach(risk => {
    if (risk.severity === '高') score -= 30
    else if (risk.severity === '中') score -= 15
    else score -= 5
  })
  return Math.max(0, score)
}
```

### 搜索功能
```javascript
function searchMaterial(keyword) {
  return furnitureData.filter(item => {
    return item.data.material_type.includes(keyword) ||
           item.data.visual_cues.some(cue => cue.includes(keyword))
  })
}
```
