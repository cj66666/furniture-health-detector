# 健康风险数据库使用指南

## 项目概述

已成功爬取并整理三大类健康风险数据:

### 1. 家具材质数据 (27条)
包含:
- **人造板类**: 刨花板、密度板、大芯板、多层实木板、指接板、OSB板
- **实木类**: 橡木、胡桃木、松木、榉木、橡胶木、水曲柳
- **皮革类**: 头层牛皮、二层皮、PU皮、科技布
- **布类**: 亚麻布、绒布
- **金属类**: 铁艺、不锈钢、铝合金
- **高分子材料**: PVC、PP、PE、ABS、PC、PS

每种材质包含:
- 化学成分(CAS号)
- 风险等级(低/中/高)
- 健康建议
- 视觉特征
- 认证标准(E0/E1/CARB P2/FSC等)

### 2. 衣料材质数据 (14条)
包含:
- **天然纤维**: 棉、麻、丝、羊毛
- **化学纤维**: 涤纶、尼龙、氨纶、粘胶纤维
- **再生纤维**: 莫代尔、天丝、竹纤维
- **混纺面料**: 真丝混纺、涤棉混纺、棉麻混纺

每种面料包含:
- 成分组成
- 化学处理(漂白、染色、阻燃等)
- 健康风险(塑化剂、PFAS、异氰酸酯等)
- 健康益处
- 护理说明
- 认证(OEKO-TEX/GOTS/GRS等)

### 3. 食物数据 (31条)
包含:
- **谷物类**: 大米、小麦、糙米、燕麦
- **蔬菜类**: 菠菜、西红柿、西兰花、胡萝卜
- **水果类**: 苹果、香蕉、蓝莓、火龙果
- **肉类**: 猪肉、鸡肉、牛肉、羊肉
- **水产类**: 鲫鱼、三文鱼、虾
- **豆类**: 黄豆、黑豆、红豆
- **奶制品**: 牛奶
- **调味品**: 食盐
- **坚果类**: 核桃、杏仁
- **食物相克组合**: 5组常见相克数据

每种食物包含:
- 营养成分(能量、蛋白质、脂肪、碳水化合物等)
- 推荐摄入量(成人/儿童/老人)
- 健康益处
- 禁忌人群
- 相克食物及原因

## 如何使用数据

### Python 读取示例
```python
import json

# 读取家具材质数据
with open('D:/Users/86198/WeChatProjects/数据库/processed/furniture.json', 'r', encoding='utf-8') as f:
    furniture_data = json.load(f)

# 遍历数据
for item in furniture_data:
    material_type = item['data']['material_type']
    risks = item['data']['risk_points']
    print(f"材质: {material_type}")
    for risk in risks:
        print(f"  风险: {risk['type']} - {risk['severity']}")

# 读取食物数据
with open('D:/Users/86198/WeChatProjects/数据库/processed/food.json', 'r', encoding='utf-8') as f:
    food_data = json.load(f)

# 查找特定食物
for item in food_data:
    if item['data']['name'] == '菠菜':
        print(f"食物: {item['data']['name']}")
        print(f"营养成分: {item['data']['nutrients']}")
        print(f"相克食物: {item['data']['incompatible_foods']}")
```

### JavaScript/微信小程序读取示例
```javascript
// 微信小程序中读取
const fs = wx.getFileSystemManager()

// 读取家具数据
try {
  const data = fs.readFileSync(
    '/data/furniture.json',

    console.log('风险:', item.data.risk_points)
  })
} catch(e) {
  console.error(e)
}
```

python main.py --category textile

python main.py --category food

python main.py --category all --verbose

- PubChem/TOXNET (化学物质数据库)
- Greenguard (低排放认证)
- OEKO-TEX (纺织品标准)
- Healthy Materials Lab
- EWG (环境工作组)

### 衣料材质
- OEKO-TEX Standard 100
- GOTS (全球有机纺织品标准)
- Textile Exchange
- EPA TextileInfo
- Made Safe

### 食物
- USDA FoodData Central
- FDA Food Database
- 中国食物成分表
- 食物相克数据库
- 营养成分数据库

## 注意事项

1. **数据真实性**: 本数据库整理自多个权威来源,但仍需结合实际情况判断

2. **食物相克**: 多数食物相克缺乏充分科学证据,不必过度恐慌

3. **个体差异**: 每个人对材质和食物的反应不同,请结合个人情况使用

4. **定期更新**: 建议定期运行爬虫程序更新数据

5. **专业咨询**: 遇到严重健康问题请咨询专业医生

## 扩展功能
